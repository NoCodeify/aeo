## Generation Tips

1. **Run each prompt 3-4 times** - Select the best output
2. **Maintain consistency** - Use the same color palette across all visuals
3. **Check mobile preview** - Ensure visuals read well at small sizes
4. **Avoid text in images** - Add any text in post-production
5. **Keep it clean** - Premium audience prefers sophistication over flash
