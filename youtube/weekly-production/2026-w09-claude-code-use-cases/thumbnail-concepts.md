# Thumbnail Concepts: "5 Claude Code Projects Making Real Money Right Now"

## Type Analysis

**Locked Title:** "5 Claude Code Projects Making Real Money Right Now"
**Title communicates:** the count (5), the tool (Claude Code), the outcome (making real money), urgency (right now)
**Thumbnail must NOT repeat:** "5", "Claude Code", "money" as a word (but a specific dollar figure adds new info)

**1+1=3 Strategy:**
- Title says WHAT (5 projects making money)
- Thumbnail must say HOW MUCH or create PROOF/CURIOSITY about the scale
- The $100K/yr ARR from DM Champ is the strongest proof point the title withholds
