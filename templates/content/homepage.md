# Homepage Template

> **Purpose:** AEO-optimized homepage copy
> **Min Words:** 500
> **Key Rule:** First 50 words must include WHO/WHAT/WHERE/PRICE

---

## Meta Tags

**Title (60 chars max):**
```
[Brand] | [Primary Service] in [Location] | [Differentiator]
```

**Meta Description (155 chars max):**
```
[Brand] offers [service] in [location]. [Key differentiator]. [Price range or positioning]. [CTA hint].
```

## Checklist Before Delivery

- [ ] First 50 words include WHO/WHAT/WHERE/PRICE

- [ ] 500+ words total
- [ ] Meta title under 60 characters
- [ ] Meta description under 155 characters
- [ ] Key differentiators clearly stated
- [ ] CTA is clear and specific
- [ ] Reads naturally out loud
